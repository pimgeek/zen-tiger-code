# 禅意虎码 (zen-tiger-code)
## 简介

面向汉字盲打爱好者的魔改版虎码输入方案（虎码新手请勿使用）

## 说明

本方案是在虎码官方单字码表基础上修改生成的，希望实现简体中文单字盲打的输入体验。具体目标如下：

* 优先覆盖日常聊天、记笔记或写文章过程中高频使用的前 6500 个常用简体中文字，从虎码官方版中提取
* 力求每个常用字都有不与其它字相冲突的全码编码，即使隐藏输入法候选框也能凭虎码拆字规范把这个字盲打出来，具体规律如下：
  * 如果一个字由 2 个虎码字根构成，必定可以用长度为 3 的编码盲打出来
  * 如果一个字由 3 个虎码字根构成，必定可以用长度为 4 的编码盲打出来
* 在少数情况下，为了规避与其它字编码冲突，可能会引入不符合拆字规范的编码。这些新引入的编码，尽可能依据虎码官方网站中的容错码、回头码或音补等策略引入。只有极少数无理码（其中音补策略使用了小鹤双拼而非全拼）
* 禅意虎码当前的码表还存在大量不完善的地方，未来可能会做进一步修订（参照最新版虎码官方码表，增加无理码规则的详细说明，排除简码中的重码，减少无理码，不再使用小鹤双拼做音补编码等）
